# 💪 단백질 가격 비서 (Protein Price Agent) — 최종 설계도

## 📌 프로젝트 기본 정보
- **프로젝트 이름**: 단백질 가격 비서 (Protein Price Agent)
- **캡스톤 트랙**: Concierge Agents (개인 생활 도움)
- **데이터 출처**: 축산물품질평가원 공공데이터 (무료 공식 데이터)
- **제출 마감**: 2026년 7월 7일 오후 3시 59분 (한국 시간)

---

## 💡 핵심 문제 (왜 이걸 만드나?)
> "운동하면서 단백질이 필요하다.
> 닭만 먹기엔 지겹고, 소고기를 먹고 싶은데 너무 비싸다.
> 소/돼지/닭/계란 중에 지금 어떤 걸 먹는 게 가장 현명한가?"

---

## 1️⃣ 이 에이전트가 답하는 질문들

| 질문 | 어떻게 답하나 |
|---|---|
| 소고기가 왜 이렇게 비싸요? | 뉴스 검색으로 이유 설명 |
| 얼마나 올랐나요? | **오늘의 최고~최저 가격 범위 안에서 지금 가격이 어디쯤 있는지** (저렴한 편/보통/비싼 편) + **최근 30일 평균과 비교해서 몇 % 높은지/낮은지**로 보여주기 |
| 지금 뭘 먹는 게 제일 현명해요? | 소/돼지/닭/계란 단백질 가격 비교 후 추천 |

> **📝 변경 메모(7/2)**: 처음엔 "평년 가격과 비교"하려 했지만, 실제 API에는 안정적으로 쓸 수 있는 평년가격 필드가 확인되지 않아 **"오늘 하루 안에서의 최고~최저 범위 위치"**로 방식을 바꿨습니다.
>
> **📝 변경 메모(7/2, 추가)**: "최근 30일 평균 대비" 지표를 새로 추가했습니다. 소/돼지는 최근 30일치 가격을 부위별로 모아 평균을 계산합니다 — 소만 물으면 약 120회(4부위×30일), 돼지만 물으면 약 90회(3부위×30일), 소와 돼지를 함께 비교하면 약 210회의 API 호출이 필요합니다. 닭/계란은 원래도 기간 조회가 가능해 추가 비용 없이 30일 평균을 계산할 수 있었습니다. 이 정도는 무료 개발 계정의 일일 호출 한도를 고려해도 데모 사용에는 충분합니다. **전년 동기 대비**는 하루 API 호출 한도 안에서 계산하기 어려워 이번 버전에서는 제외했습니다. (추후 1년치 가격을 API로 미리 수집해 CSV로 저장해두는 방식으로 추가 가능)

---

## 2️⃣ 절대 하면 안 되는 일 (가드레일)

- ❌ 공식 데이터(축산물품질평가원) 이외의 가격 정보는 절대 쓰지 않기
- ❌ "앞으로 정확히 몇 월에 내릴 거예요" 같은 확정적 미래 예측은 하지 않기
- ❌ 특정 마트의 실시간 가격 비교는 하지 않기 (데이터 접근 불가)
- ❌ 단백질 식품 이외의 질문에는 답하지 않기
- ❌ API 키(비밀번호)를 코드 안에 직접 쓰지 않기

## 3️⃣ 쓸 도구 목록

| 도구 | 역할 | 기술 |
|---|---|---|
| 🔌 축산물품질평가원 API (`get_protein_prices`) | 소/돼지/닭/계란 오늘 가격 가져오기 | ADK Function Tool |
| 🔌 뉴스 검색 도구 (`news_tool`) | "왜 비싼지" 이유 찾기 | ADK `google_search` + `AgentTool`로 감싼 서브 에이전트 |
| 🔌 가격 분석 도구 (`analyze_price`) | 오늘 시세 범위 안에서 저렴/보통/비쌈 판정 + 최근 30일 평균 대비 상승/하락 판정 | ADK Function Tool |

> **📝 변경 메모(7/2)**: 처음 계획한 "MCP 서버"는 실제로는 사용하지 않았습니다. 대신 뉴스 검색은 별도 에이전트(`news_search_agent`)를 만들고, 이를 `AgentTool`로 감싸서 메인 에이전트의 도구 중 하나로 등록하는 **멀티 에이전트 도구 조합(agent-as-tool)** 방식으로 구현했습니다. (`google_search`는 다른 도구와 함께 못 쓰는 제약이 있어, `sub_agents` 대신 이 방식을 택했습니다.)

## 4️⃣ 캡스톤 기술 6가지 중 실제 증명한 것들

| 기술 | 충족 방법 | 증명 위치 |
|---|---|---|
| ① 에이전트 (ADK) | `root_agent`가 여러 도구를 판단해서 조합 호출 | 코드 (`app/agent.py`) |
| ② 멀티 에이전트 도구 조합 | `news_search_agent`를 `AgentTool`로 감싸 메인 에이전트 도구로 사용 | 코드 (`app/agent.py`) |
| ③ Antigravity | 코드 짤 때 Antigravity IDE 사용 | 영상 |
| ④ 보안 기능 | API 키를 `.env` 파일에 숨겨두고 `.gitignore`로 보호 | 코드 |
| ⑤ 배포 가능성 | Vertex AI Agent Engine / Cloud Run 배포 가능한 구조 | 영상 |
| ⑥ Skills | `skills/SKILL.md` 파일로 개발 규칙 문서화 | 코드 |

(캡스톤 제출 기준은 6개 중 최소 3개 증명이며, 위 6개를 모두 코드 또는 영상으로 다룹니다.)

> 📊 Today's Price: 등심 11,819원 (최고 13,016원, 최저 10,068원), 설도 4,721원, 안심 14,642원, 양지 6,424원
>
> 📈 오늘 시세 범위 내 위치: 등심 59.4%(보통), 설도 48.6%(보통), 안심 42.3%(보통), 양지 54.7%(보통)
>
> 📅 최근 30일 평균 대비: 등심 +3.2%(평균과 비슷함), 설도 -6.1%(30일 평균보다 낮음)
>
> 📰 Why: 사육 마릿수 감소세 지속과 명절 수요, 수입 소고기 가격 상승에 따른 대체 효과로 도매가 상승
>
> 💡 Recommendation: 오늘 기준 소/돼지/닭/계란 중 가장 저렴한 편인 단백질은 돼지고기 앞다리입니다

> **📝 변경 메모(7/2)**: 위 예시는 30일 평균 대비 항목이 추가되기 전 실제 테스트 결과에, 새 항목("📅 최근 30일 평균 대비")이 어떻게 추가되는지 형식만 보여주기 위해 수치를 예시로 덧붙인 것입니다. (전년 동기 대비 항목은 이번 버전에 포함되지 않음)

## 6️⃣ 보안 설계

- API 키는 `.env` 파일에 저장 (코드에 직접 쓰지 않음)
- `.env` 파일은 `.gitignore`에 추가 (깃허브에 절대 올라가지 않도록)
- 코드에는 `os.getenv("LIVESTOCK_API_KEY")` 방식으로 불러오기
- `python-dotenv`의 `load_dotenv()`로 실행 시점에 안전하게 로드
